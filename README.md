# A4P Platform V7 corrigé

## Objectif
Version **visible immédiatement** sur GitHub Pages.

## Contenu
- Accueil
- Dashboard club
- Équipes
- Joueurs
- Passations
- Données démo dans `data/club-data.json`

## Déploiement GitHub
1. Créer un nouveau dépôt GitHub.
2. Déposer tout le contenu du dossier à la racine.
3. Activer GitHub Pages sur `main` / root.
4. Ouvrir `index.html`.

## Important
- Cette V7 est une **version visible et statique**.
- La V8 / V9 sont des architectures SaaS qui ne sont pas destinées à GitHub Pages seules.
